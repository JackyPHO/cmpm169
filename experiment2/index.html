Experiment 2 - Vector Art, Animation & Interactivity

Description

In this experiment, I am testing out p5.js to create art designs that utilize animations and vector shapes.

Technical

Imitate: For the general shape, I took a code snippet that would have a transforming lines of circle that will change based on mouse position. The setup() and draw() were copied from the reference design. Reference
Integrate: I used a color changing property for shapes that I would add into my transforming circle. The HSB color value changes based on mouse position.Reference
Innovate: For my creativity, I would create disappearing circles from the reference design. A push and pop method from ChatGPT helped with creating multiple instances of the shape will keep the same behaviors of color change and appearance.Reference
